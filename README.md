# Корнуэльская комната
Создать сцену: Корнуэльская комната обязательно в виде большого куба (все грани должны быть). Внутри как минимум 1 объект (не прозрачный и не отражающий. На выбор куб или шар. Один точечный источник света. - 4 балла

<img width="480" height="360" alt="image" src="https://github.com/user-attachments/assets/804459f4-126d-4123-a884-93f93f68c61e" />

1) За 2-й объект того же вида + 1 балл.
2) За дополнительные объекты другого вида ( шар или куб) ещё + до 2-х баллов (1 доп объект -1 балл, 2 объекта - 2 балла).
3) Добавление зеркального отражения для объекта + до 2-х баллов (по 1-му за каждый тип объекта). С возможностью включать и выключать зеркальность.
4) Добавление прозрачности + до 2-х баллов (по 1-му за каждый тип объекта). С возможностью включать и выключать прозрачность.
5) Стена комнаты зеркальная с возможностью выбора стены +1 балл.
6) Добавление дополнительного источника света (положение можно менять)+ 1 балл.
   
Все баллы выше 8 идут в бонусные баллы к экзамену.
